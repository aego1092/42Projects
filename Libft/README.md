# Libft

## Description
This project consists of creating a C library containing a collection of functions. Some standard standard libc, BSD libc functions functions are re-coded by creator, in addition to other functions.
The goal is the improvement of student knowledge, as Libft project force to understand the logic of tool, the memory management and hardware, and to critical thinking needed to address edge cases.
Testing by doing main function is one of the best way to gain knowledge

## Instructions
Compile the library with `make`.
command list:
make          # compile libft.h and return object (.o) files and the archive libft.a
make clean    # remove object (.o) files
make fclean   # remove object files and the archive libft.a
make re       # is sole command for doing make fclean and make
